Page Pipeline.

An automated multi-channel campaign sequencing platform connecting sales, marketing, and lead workflows.

Overview

Managing outreach campaigns manually is a persistent bottleneck for sales divisions. Reaching out via LinkedIn, tracking replies, and coordinating follow-up email sequences in parallel consumes hours of calendar time. We built Page Pipeline to organize multi-step campaign sequences into an automated, event-driven compiler pipeline.

Project Type

Custom Software

Timeline

4 Weeks

Core Outcome

68% Open Rates

Client State

B2B Services Agency

The Problem

Operators spent up to 18 hours weekly managing disconnected lead spreadsheets, sending manual connection requests, typing template updates, and tracing replies. Delivering messages on single channels triggered high spam scores and low conversion rates.

The Solution & Engineering Process

We designed an automated campaign sequence runner. The engine connects to LinkedIn APIs and Gmail REST endpoints, executing triggers and mapping conversions in a live analytics dashboard. Below is the active data pipeline:

Tech Stack

Node.js LinkedIn API Gmail API Redis Cache PostgreSQL

Key Features

  • Campaign Sequencer: Visual drag-and-drop workflow configuration.
  • Jitter Delays: Randomized automation offsets protecting domains.
  • Analytics Dashboard: Live logs showing opens, clicks, and reply stats.

Engineering Challenges

Domain deliverability is high-risk during outreach. Sending campaigns rapidly flags domains. We resolved this by building customizable, randomized jitter schedules in Redis, ensuring triggers execute at offsets between 30 seconds and 3 minutes to look like human interactions.

Outcome

By moving to automated sequencing, campaign open rates stabilized at 68%, with reply rates climbing to 22%. Sales divisions expanded pipelines with zero manual formatting delays.
