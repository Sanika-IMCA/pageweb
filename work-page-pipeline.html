Page Pipeline.

An automated sequence pipeline editor that maps and schedules sales prospects across email, LinkedIn, and social networks.

View GitHub Repository

Overview

Effective lead generation requires persistent follow-up across multiple platforms. However, founders and sales representatives lose hours daily switching tabs, copy-pasting sequences, and manually updating CRM logs. We built Page Pipeline to centralize and automate multi-channel follow-up sequences in a single visual editor.

The Problem

Siloed outreach tools cause overlapping lead databases, broken message sync, and missed follow-up tasks. Reps struggle to align Connection requests on LinkedIn with cold email sequences, leading to cold, disjointed conversations.

Discovery Workshop

We realized that visual pipelines help reps conceptualize prospect journeys. Giving them a node-based interface to build and schedule connection events, email sequences, and delay rules in one flow significantly improves campaign structures.

Product Thinking

Instead of building standard list-based campaign tables, we structured the interface as a flow builder graph. This matches the way sales managers conceptualize journeys ("If accepted -> send connection message, else -> send email"). Visualizing outreach as pathways increases user understanding and layout organization.

Design & Wireframes

We prioritized custom design tokens that support active and inactive states for nodes. The user flows map leads entering the pipeline, moving between nodes based on response flags, and converting on metrics panels.

System Architecture

The engine runs a visual flowchart builder that translates graph nodes into active cron schedules. These schedules manage connection requests, message prompts, and response listeners. Telemetry listeners query active states to update CRM parameters automatically.

Tech Stack

React Flow Node.js BullMQ Queue Redis Cache PostgreSQL LinkedIn Scraper API SES Mail Server

Key Features

  • ⚡ Node-Based sequence builder: Drag-and-drop actions, delays, and split triggers.
  • ⚡ Cross-Channel Sync: Records activity logs across email and socials.
  • ⚡ Prospect profile cards: Visual records of prospect metrics.
  • ⚡ Sequence Conversion tracking: Real-time charts of open, click, and reply rates.

Engineering Challenges

Rate limits on social network connections are highly restrictive. Exceeding thresholds risks flagging domain profiles. We resolved this by configuring randomized jitter delays, batching messages dynamically, and prioritizing high-intent warm lists.

Outcome

Clients achieved average open rates of 68% and reply rates of 22%. Integrating visual flows improved outreach team efficiency, saving reps an average of 12 hours weekly.

Future Roadmap

We are configuring contextual AI triggers. By scraping target prospect bios, the compiler will draft customized icebreaker statements for connection notes.
